Add Colombian states to WooCommerce

== Description ==

A list of colombian states (departamentos) will be added to WooCommerce to be rendered as a drop down list instead of an input text

== Installation ==

1. Upload to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy
